# APP TEAMBOX

## Descripción
**TeamBox** es una aplicación diseñada para registrar y clasificar a los boxeadores amateurs en España según su categoría y peso.

## Objetivo
El objetivo principal de **TeamBox** es proporcionar un acceso rápido y eficiente a una base de datos de deportistas del panorama nacional. Esto permite:
- Organizar eventos deportivos de manera ágil.
- Realizar emparejamientos de combate de forma rápida.
- Encontrar sustitutos en caso de incomparecencia o problemas burocráticos que impidan la participación de un boxeador.

Con **TeamBox**, los organizadores pueden optimizar la gestión de competiciones y garantizar el desarrollo adecuado de los eventos.
